#  Info

A hands-on repository for learning cybersecurity by building security tools from scratch.

## 📁 Repository Structure

- **`tools/`** — Security tools built from scratch (Python, php)
- **`environment_setup/`** — Base environment setup  needed before starting any project here
- **`base-knowledge/`** — General theoretical and scientific cybersecurity knowledge/notes

## 🛠️ Projects

### hash-identifier
Python CLI tool that identifies hash types (MD5, SHA family, bcrypt, NTLM, and more) using a data-driven rules engine (`rules.json`).

## 🎯 Focus Areas

- Blue Team / SOC fundamentals
- Network Security
- Security tooling with Python
